## 模块职责

`book/` 目录存放围绕 `restored-src/src/` 源码整理的学习型电子书，共 20 篇章节，分 4 个分卷。电子书侧重"业务流程 + 源码映射 + 复刻抽象"，帮助读者从 CLI 入口到多代理协同建立完整的系统认知。

---

## 分卷结构

| 分卷 | 覆盖章节 | 主题 |
|---|---|---|
| `part-1-主业务流/` | 01-06 | CLI 启动 -> 初始化 -> 会话模型 -> query 主循环 -> 工具编排 -> 输出收尾 |
| `part-2-扩展能力流/` | 07-10 | MCP 接入 -> Skills -> Plugins/Hooks -> 权限与安全边界 |
| `part-3-远程协同流/` | 11-14 | Bridge 主链路 -> Remote Session -> 后台会话 -> 多代理协同 |
| `part-4-附录/` | 90-99 | 源码目录地图 -> 核心文件索引 -> 关键类型抽象 -> 练习题 |
