# Phase 5 Plan 02

## Objective

把 discovery 数据层落成真正可进入的 `/discover` 页面，并把首页、导航和关键板块页的入口统一收口到这个发现中心。

## Tasks

### Task 1: 组装 `/discover` 页面与共享发现组件

**Files:**
- Create: `.vitepress/theme/components/DiscoveryTypeBadge.vue`
- Create: `.vitepress/theme/components/DiscoveryGoalRoutes.vue`
- Create: `.vitepress/theme/components/DiscoveryStartGrid.vue`
- Create: `.vitepress/theme/components/DiscoveryTopicHub.vue`
- Modify: `.vitepress/theme/index.ts`
- Create: `docs/discover/index.md`
- Modify: `scripts/check-discovery-experience.mjs`

- [ ] Step 1: 创建统一内容类型标签组件，保证章节、实践项目、学习路线和进阶专题的前台语言一致
- [ ] Step 2: 创建按目标选路线与推荐起点组件，确保首屏能先帮用户做第一次选择
- [ ] Step 3: 创建主题聚合组件，把理论、实践和中级专题按主题收口到同一页中
- [ ] Step 4: 组装 `docs/discover/index.md`，让页面具备“首屏选路线 -> 推荐起点 -> 按类型浏览 -> 按主题继续探索”的结构
- [ ] Step 5: 扩展 `check-discovery-experience`，验证 `/discover` 关键区块和共享组件已接入
- [ ] Step 6: 运行 `bun run build:strict`

- `/discover` 已存在并以“按目标选路线”为首屏主轴
